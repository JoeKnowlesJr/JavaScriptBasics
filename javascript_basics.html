JavaScript Basics
JavaScript is a cross-platform, object-oriented scripting language. JavaScript is extremely popular for a variety of reasons. Iy is a small, and lightweight language, allowing maximum flexibility for developers to take it in a bunch of different directions. JavaScript lives inside a host environment (a web browser or Node server), it can be connected to the objects of these environments to provide programmatic control over them.

You can use conditional statements in your code to acomplish this. There are the following conditional statements: if a specified condition is true, do this code in our code block; else if to specify a new condition to test, if the condition is false; else we execute this block of code
  • Loops There are namy different kinds of loops in every programming language, but they all essentially do the same thing: they will repeat an action some number of times. Imagine you have to run a mile, well you run around the track four times and then you stop. That's a loop!
